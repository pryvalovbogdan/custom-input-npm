# What is this?

Create input with cross button by id.

# Installation

`npm i custom-npm-input --save`

# Usage

import { inputWithCross } from 'custom-npm-input';

inputWithCross({
    id: 'input',
    onFocus: () => {},
    color: 'red',
});

## Options

*    *id*: 'input',
*    *onFocus*: () => {},
*    *color*: 'red',
